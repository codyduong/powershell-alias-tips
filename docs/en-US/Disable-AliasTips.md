# Disable-AliasTips

## SYNOPSIS
Disables alias-tips

## SYNTAX

```
Disable-AliasTips
```

## DESCRIPTION
Disables alias-tips by setting $env:ALIASTIPS_DISABLE to $true

## EXAMPLES

### EXAMPLE 1
```
Disable-AliasTips
```

## INPUTS

### None. This function does not accept any input.

### None. This function does not accept any input.
