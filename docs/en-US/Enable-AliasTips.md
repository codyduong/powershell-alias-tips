# Enable-AliasTips

## SYNOPSIS
Enables alias-tips

## SYNTAX

```
Enable-AliasTips
```

## DESCRIPTION
Enables alias-tips by setting $env:ALIASTIPS_DISABLE to $false

## EXAMPLES

### EXAMPLE 1
```
Enable-AliasTips
```

## INPUTS
