# Enable-AliasTips

## SYNOPSIS
Enables alias-tips

## SYNTAX

```
Enable-AliasTips [<CommonParameters>]
```

## DESCRIPTION
Enables alias-tips by setting $env:ALIASTIPS_DISABLE to $false

## EXAMPLES

### EXAMPLE 1
```
Enable-AliasTips
```
